# API

Supabase backend services for WorkOrder.

## Overview

This directory contains the Supabase configuration and infrastructure code for WorkOrder's backend services. It includes database migrations, Edge Functions, and Supabase configuration.

## Structure

- `supabase/` - Supabase CLI configuration and migrations
- `supabase/migrations/` - Database migration files
- `supabase/functions/` - Edge Functions (future)
